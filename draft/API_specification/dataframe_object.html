    Dataframe object¶

    A conforming implementation of the dataframe API standard must provide and support a dataframe object having the following attributes and methods.

    Operators¶

    A conforming implementation of the dataframe API standard must provide and support a dataframe object supporting the following Python operators.

    Arithmetic Operators¶

    A conforming implementation of the array API standard must provide and support an array object supporting the following Python arithmetic operators.

    • x1 + x2: DataFrame.__add__()

      • operator.add(x1, x2)

      • operator.__add__(x1, x2)

    • x1 - x2: DataFrame.__sub__()

      • operator.sub(x1, x2)

      • operator.__sub__(x1, x2)

    • x1 * x2: DataFrame.__mul__()

      • operator.mul(x1, x2)

      • operator.__mul__(x1, x2)

    • x1 / x2: DataFrame.__truediv__()

      • operator.truediv(x1,x2)

      • operator.__truediv__(x1, x2)

    • x1 // x2: DataFrame.__floordiv__()

      • operator.floordiv(x1, x2)

      • operator.__floordiv__(x1, x2)

    • x1 % x2: DataFrame.__mod__()

      • operator.mod(x1, x2)

      • operator.__mod__(x1, x2)

    • x1 ** x2: DataFrame.__pow__()

      • operator.pow(x1, x2)

      • operator.__pow__(x1, x2)

    Arithmetic operators should be defined for a dataframe having real-valued data types.

    Comparison Operators¶

    A conforming implementation of the dataframe API standard must provide and support a dataframe object supporting the following Python comparison operators.

    • x1 < x2: DataFrame.__lt__()

      • operator.lt(x1, x2)

      • operator.__lt__(x1, x2)

    • x1 <= x2: DataFrame.__le__()

      • operator.le(x1, x2)

      • operator.__le__(x1, x2)

    • x1 > x2: DataFrame.__gt__()

      • operator.gt(x1, x2)

      • operator.__gt__(x1, x2)

    • x1 >= x2: DataFrame.__ge__()

      • operator.ge(x1, x2)

      • operator.__ge__(x1, x2)

    • x1 == x2: DataFrame.__eq__()

      • operator.eq(x1, x2)

      • operator.__eq__(x1, x2)

    • x1 != x2: DataFrame.__ne__()

      • operator.ne(x1, x2)

      • operator.__ne__(x1, x2)

    Comparison operators should be defined for dataframes having any data type.

    Methods¶

    DataFrame.__add__(other)

    Add other dataframe or scalar to this dataframe.

    DataFrame.__eq__(other)

    Compare for equality.

    DataFrame.__floordiv__(other)

    Floor-divide (returns integers) this dataframe by other dataframe or scalar.

    DataFrame.__ge__(other)

    Compare for "greater than or equal to" other.

    DataFrame.__gt__(other)

    Compare for "greater than" other.

    DataFrame.__le__(other)

    Compare for "less than or equal to" other.

    DataFrame.__lt__(other)

    Compare for "less than" other.

    DataFrame.__ne__(other)

    Compare for non-equality.

    DataFrame.__mod__(other)

    Return modulus of this dataframe by other (% operator).

    DataFrame.__mul__(other)

    Multiply other dataframe or scalar with this dataframe.

    DataFrame.__pow__(other)

    Raise this dataframe to the power of other.

    DataFrame.__sub__(other)

    Subtract other dataframe or scalar from this dataframe.

    DataFrame.__truediv__(other)

    Divide this dataframe by other dataframe or scalar.
